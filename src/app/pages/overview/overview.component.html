Project Overview

Bedrock VM/AP Analytics Portal

Centralized, scalable, and secure Snowflake-native portal replacing slow, manual Excel workflows for VM/AP analysis.

Snowflake Audit-ready SSO + RLS Exports

Share with stakeholders or attach to the steering deck.

Purpose

Replace slow, manual, Excel-based VM/AP analysis with a centralized, secure, and scalable Snowflake-powered portal. Detect duplicate vendors/invoices quickly and consistently, access granular historical spend without file transfers, generate audit-ready reports from live Snowflake data, and stand up a reusable Central Vendor System.

Background & Current Challenges

  • Manual workflow: heavy dependence on ad-hoc SQL and Excel per engagement.
  • Excel dependency: repeat build of structures/reports; version confusion and data loss.
  • High data volume: ERP extracts too large for Excel; incomplete AP history and slow analysis.
  • Limited reuse: hard to reuse duplicate knowledge or risk signals across clients.
  • Human error and weak lineage when preparing audit files.
  • Security risks from emailed vendor/payment data and potential PII exposure.
  • Scalability issues with different ERP logic and no standardized automation.
  • Lack of transparency: no permanent audit trail, weak defensibility, and inefficient use of data team time.

Why build the portal

  • Centralize data in Snowflake as a single source of truth.
  • Enable real-time duplicate detection, spend metrics, and audit dashboards.
  • Provide self-service analytics to auditors/managers without SQL.
  • Cut processing time and manual errors by 60%+.
  • Ensure traceability, governance, and security compliance.

Master Vendor Repository (cross-client)

  • Consolidates and enriches vendor identities across clients plus selected public sources.
  • Builds a validated global dataset (name, tax ID, address, contact) to reduce redundancy.
  • Tags vendors as Validated, Pending Validation, or Client-Specific; keeps client transactions siloed.
  • Accelerates onboarding and delivers consistent, cross-client reporting with better risk checks.

Non-technical summary

Secure, instant access to live vendor and AP data, no Excel/email file sharing, interactive dashboards with filtering, commenting, exporting, and full audit logging.

“No more Excel chaos, no more SQL re-runs—just clean, auditable data available 24/7.”
